# cpp
C++ Code Examples

## Compilation

To complie the program GNU C++ compiler g++ can be used as shown below in terminal.

```bash
g++ main.cpp -o prog -std=C++17
```
To run the program in Linux system like Ubuntu execute as shown below.

```bash
./prog
```
